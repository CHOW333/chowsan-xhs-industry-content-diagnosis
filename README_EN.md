# ChowSan XHS Industry Content Diagnosis

Public Codex skill for diagnosing how an industry or local business should create Xiaohongshu content.

## Skill

```text
chowsan-xhs-industry-content-diagnosis
```

## What It Does

This skill helps generate a practical Xiaohongshu content diagnosis for a business category, store, local service, or industry.

It outputs:

- Industry diagnosis.
- User search intent.
- What the business should stop posting.
- Content directions.
- Title formulas.
- Keyword layout.
- 7-day content plan.
- Store visit, consultation, community, or repeat-purchase bridge.

## Install

Copy the folder below into your Codex skills directory:

```text
chowsan-xhs-industry-content-diagnosis/
```

Typical local destination:

```text
~/.codex/skills/chowsan-xhs-industry-content-diagnosis
```

## Example Prompt

```text
Use $chowsan-xhs-industry-content-diagnosis to diagnose how a community fruit store should create Xiaohongshu content.
```

## Privacy

This is the public version. It does not include private local paths, private social posts, or personal Obsidian vault settings.
